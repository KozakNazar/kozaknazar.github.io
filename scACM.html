Питання 3.1

Для алгоритму, що заданий за допомогою функції мовою С, розрахувати обчислювальну складність для «найгіршого випадку».

void bulbSort(int * array, int leftIndex, int rightIndex){

    if(!array){

        return;

    }

    ++rightIndex;        

    while(leftIndex < --rightIndex){

        for(unsigned int index = leftIndex; index < rightIndex; ++index){

            if(array[index] > array[index + 1]){

                int temp = array[index];

                array[index] = array[index + 1];

                array[index + 1] = temp;

            }

        }   

    } 

}


  • O(n*n)

  • O(n)

  • O(2n)

  • O(4n)
